## Explorador de Carpetas 📁

**Analiza y visualiza la estructura de tus carpetas de forma sencilla.**

Este proyecto te permite explorar y visualizar la estructura de carpetas, incluyendo sus archivos, de una manera intuitiva y fácil de entender. Simplemente selecciona una carpeta y la aplicación generará una representación visual de su estructura, junto con un resumen detallado del contenido de cada archivo. 

**¡Obtén el contexto crucial para darselo a tu IA favorita!** 🚀  Entender la estructura y el contenido de un proyecto es fundamental para alimentar y entrenar modelos de IA de manera efectiva. Esta herramienta te permite extraer información valiosa sobre la organización de los datos, las relaciones entre archivos y el código fuente, lo que facilita la creación de datasets y la optimización del rendimiento de la IA.

## Características ✨

* **Visualización de la estructura de carpetas:** Genera una representación visual en forma de árbol de la estructura de la carpeta seleccionada.
* **Resumen del contenido de los archivos:**  Muestra el contenido de cada archivo de texto (como .txt, .js, .css, etc.) dentro de la estructura.
* **Descarga de la estructura:** Permite descargar la estructura de la carpeta y el contenido de los archivos en un archivo de texto (.txt).
* **Soporte para .gitignore:**  Respeta las reglas definidas en el archivo .gitignore para excluir archivos y carpetas de la visualización.
* **Interfaz de usuario intuitiva:**  Diseño simple y fácil de usar con modo oscuro y claro.
* **Responsivo:** Se adapta a diferentes tamaños de pantalla.


## Cómo usar 🛠️

1. **Selecciona una carpeta:**  Haz clic en el botón "Seleccionar Carpeta" y elige la carpeta que deseas explorar.
2. **Visualiza la estructura:**  La aplicación mostrará la estructura de la carpeta en forma de árbol y el contenido de los archivos de texto.
3. **Descarga la estructura:** Haz clic en el botón "Descargar" para guardar la estructura de la carpeta y el contenido de los archivos en un archivo de texto.


## Tecnologías utilizadas 📦

* **React:**  Librería de JavaScript para construir interfaces de usuario.
* **Lucide React:**  Librería de iconos para React.
* **HTML:**  Lenguaje de marcado para estructurar el contenido de la página web.
* **CSS:**  Lenguaje de hojas de estilo para dar formato a la página web.
* **JavaScript:**  Lenguaje de programación para la lógica de la aplicación.
